# 运维内网管理系统 — 项目总览

> 版本：V2.0 | 日期：2026-07-14

---

## 一、项目定位

企业内部运维内网统一管理平台，对标阿里云/腾讯云控制台体验，对 6 台 Dell R740 上的 KVM/Ceph/K8s/中间件实现**统一纳管、可视化管理、自助化交付**。

## 二、技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 前端 | React 18 + Ant Design Pro + TypeScript | Vite 构建 |
| 后端 | Go 1.22 + Gin + GORM | RESTful API |
| Agent | Go 轻量 Agent | HTTP + WebSocket |
| 数据库 | MySQL 8.4 | ops_platform 库 |
| 缓存 | Redis 7 | 缓存 + Token 黑名单 + 消息队列 |
| 监控 | Prometheus + Grafana | 平台自带 |
| 仓库 | GitHub | github.com/jkm007/ |

## 四、一期 MVP 范围

| 模块 | 核心功能 |
|------|---------|
| 权限与审计 | 登录/用户/角色/多租户/RBAC/操作审计 |
| 计算 | 物理机纳管 + KVM VM 全生命周期（7步向导：宿主机→镜像→cloud-init→规格→存储类型→网络→确认） |
| 存储 | Ceph RGW 对象存储 Bucket + Ceph Dashboard + ��份管理 |
| 网络 | 虚拟网络 + 安全组 + CLB + 浮动IP |

### 存储策略

创建 VM 时用户自由选择：
- **本地 qcow2**：高性能，路径 `/home/kvm/images/`，不支持在线迁移
- **Ceph RBD**：高可用，可在线迁移

### 网络策略

一期采用 **br0 桥接 + 物理交换机 VLAN Trunk**（需交换机支持），备选 VXLAN overlay。
